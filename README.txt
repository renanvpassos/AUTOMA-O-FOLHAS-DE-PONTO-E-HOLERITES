O programa consiste em realizar o disparo das folhas de pontos de uma unica forma, em uma unica plataforma, para evitar ter que ficar criando e-mail por e-mail. 

No código já estão pré-configurados os e-mails dos colaboradores e também dados como e-mail e senha de acesso ao e-mail que será o remetente. Que puxa a informação do "script_gerar_env.py" localizado na pasta. 


O código está simples porém 100% funcional e atendeu as necessidades estabelecidas.
